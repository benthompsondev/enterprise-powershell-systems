# Code Of Conduct

This repo is for practical IT automation, sanitized examples, and learning from real operational patterns.

Please keep discussion respectful, specific, and privacy-safe.

Good feedback looks like:

- "This run step is unclear."
- "This example output could be easier to follow."
- "This validation check should catch one more bad row."
- "This wording might imply the script changes live systems when it only simulates them."

Please do not post private workplace data, real user data, internal screenshots, secrets, or attack instructions. Keep examples fake and keep feedback useful.

If something sensitive is accidentally shared, point out the category only and avoid repeating the value.
